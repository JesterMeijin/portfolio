Description

Dungeon Quest is a little game where an Adventurer, looking for treasures in a dark and mysterious maze, is hunted by Hunters who wants to protect their belongings.  To achieve their objectives, players will evolve in this labyrinth with the only help of a torch. 

It was created for a school project at Polytech Nantes. The objective was to modify and improve an existing code in Python, already implementing an HTTP server to play on a local network (LAN).

We have created a new interface, created a labyrinth with wall sprites, created an animated walk with character sprites, added server requests for the position and orientation of the players, added a torch mechanism to explore the labyrinth and finally, added the objectives of the game...

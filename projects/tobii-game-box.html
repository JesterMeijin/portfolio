Tobii game box

PreviousNext

Description

This project was a school project of Polytech Tours where the problematic was to create a program to help disabled people to entertained themselves or to train their ability to see with the only use of their eye using the Tobii Eye Tracker 4C device.

The only option that was offered to us to take advantage of this device (get the eye position on the screen for example), was the Tobii Stream Engine, a low-level SDK developped by Tobii in C / C++.
After some research, the only advanced library to create user interfaces available at this time was SDL2.

Despite some complication during this all project, we manage to create the main architecture of the Tobii Game Box with a first mini-game called Remember. In this game, the player needs to remember the position and the number of different circle on the screen during the first phase, and point at them in the right order in the second phase where the number disapears.
